# 取名 - 智能取名助手

## 用途

根据用户需求智能生成名字，支持多种场景：
- **宝宝取名**：基于生辰八字、五行补缺、诗词典故
- **公司/品牌起名**：行业属性、易记易传播、寓意吉祥
- **游戏ID/网名**：酷炫、古风、二次元、英文适配
- **笔名/艺名**：文艺气质、独特记忆点

## 使用时机

当用户询问以下问题时触发：
- "帮我取个名字"
- "给宝宝取名字"
- "公司起名"
- "游戏ID推荐"
- "取个笔名"
- "这个名字怎么样"
- 任何与取名、命名相关的请求

## 工作原理

### 核心流程

1. **识别场景**
   - 分析用户需求，确定是宝宝取名/公司起名/游戏ID/笔名
   - 不同场景启用不同的规则和权重

2. **收集信息**
   - 宝宝取名：姓氏、性别、出生年月日时辰、期望寓意、避讳字
   - 公司起名：行业、品牌定位、目标受众、中英文需求
   - 游戏ID：风格偏好（古风/二次元/英文/霸气）、字数要求
   - 笔名：创作领域、气质风格、记忆点要求

3. **智能生成**
   - 调用 `name_generator.py` 生成候选名字
   - 混合古诗词精选 + 智能组合算法
   - 根据场景应用不同权重

4. **多维评分**
   - 五行匹配（25%）：生辰八字缺什么补什么
   - 寓意深度（20%）：诗词出处、文化内涵
   - 音韵和谐（20%）：平仄搭配、朗朗上口
   - 笔画吉凶（15%）：五格剖象、书写美观
   - 独特性（10%）：常见度评估
   - 结构平衡（10%）：字形搭配

5. **质量检查**
   - 谐音检查：过滤贬义谐音组合
   - 避讳检查：排除用户指定的避讳字
   - 生僻字检查：避免过于生僻难以辨认

6. **输出报告**
   - 按综合评分排序，展示5-8个精选名字
   - 每个名字包含详细解析和各项指标

### 评分算法

```
综合评分 = 
  五行匹配 × 25% +
  寓意深度 × 20% +
  音韵和谐 × 20% +
  笔画吉凶 × 15% +
  独特性 × 10% +
  结构平衡 × 10%
```

每项0-100分，满分100分。

## 核心模块

### name_generator.py - 名字生成引擎

```python
from scripts.name_generator import NameGenerator

generator = NameGenerator(
    surname="王",
    gender="male",
    birth_datetime="2024-03-15 09:00",
    style="classical",
    meanings=["wisdom", "talent"]
)

results = generator.generate(count=8)
# 返回: [{"name": "王子墨", "score": 96, "details": {...}}, ...]
```

### bazi_analyzer.py - 八字五行分析

```python
from scripts.bazi_analyzer import BaziAnalyzer

analyzer = BaziAnalyzer()
bazi = analyzer.calculate("2024-03-15", "09:00")
# 返回: {"year": "甲辰", "month": "丁卯", "day": "己卯", "hour": "己巳",
#        "wuxing": {"wood": 3, "fire": 1, "earth": 2, "metal": 0, "water": 0},
#        "missing": ["metal", "water"],
#        "favor": "金水"}
```

### name_scorer.py - 名字评分系统

```python
from scripts.name_scorer import NameScorer

scorer = NameScorer(bazi_result=bazi)
score = scorer.score("王子墨")
# 返回: {"total": 96, "breakdown": {"wuxing": 95, "meaning": 98, ...}}
```

### pronunciation.py - 音韵分析

分析平仄、声韵搭配、朗朗上口程度。

### stroke_analyzer.py - 笔画分析

计算五格剖象、字形结构、书写美观度。

### homophone_checker.py - 谐音检查

检查全名谐音，过滤贬义组合。

## 注意事项

### 八字计算
- 使用真太阳时计算（考虑出生地经度）
- 节气分界（立春为年分界，月干支按节气）
- 子时分早晚（23-24点为晚子时）

### 五行判断
- 字的五行属性以康熙字典为准
- 支持按笔画数计算（1-2木、3-4火、5-6土、7-8金、9-10水）
- 也有按字义判断（如"江"属水，"炎"属火）

### 隐私说明
- 所有数据本地存储，不上传服务器
- 八字计算在本地完成
- 不收集用户的个人生辰信息

### 免责声明
- 取名结果仅供参考，传统文化内容存在多种解读
- 最终取名决定权在用户
- 建议结合家族传统、个人喜好综合考虑
